# MCQ - Multiple Choice Question Practice App

A lightweight, interactive Multiple Choice Question (MCQ) practice application built with vanilla HTML, CSS, and JavaScript.


## Live
Try it out here: [https://albus-percival.github.io/mcq-practice/](https://albus-percival.github.io/mcq-practice/)

## How to Use

1. **Open** `index.html` in your web browser
2. **Enter** the number of questions you want to practice (1 or more)
3. **Click** the "Start" button
4. **Select** your answer for each question by clicking one of the options (A, B, C, or D)
5. **Clear** any answer using the delete button (trash icon) if needed
6. **Submit** your answers by clicking the Submit button
7. **Review** your results on the answers page, which shows:
   - Your answer for each question
   - Number of correct answers
   - Number of wrong answers
   - Number of unanswered questions
   - Total time taken
8. **Restart** to begin another quiz


## Browser Compatibility

Works on all modern browsers that support:
- ES6 Modules
- CSS Grid and Flexbox
- View Transitions API (optional, for smooth page transitions)
- DOM manipulation with JavaScript
